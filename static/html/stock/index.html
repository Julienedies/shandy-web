市场是不确定的

  • 市场反映一切
  • 历史会重复
  • 价格以趋势方式行进
  • 反身性
  • 价格沿最小阻力方向前进
  • 市场是投机的，市场是逐利的。
成交量
  • 放量：看空、看多的
  • 缩量：观望
行情阶段
  1. 初期，趋势不明显，，
  2. 行情中期，已经有一定涨幅，趋势明显，成交量放大，止盈
  3. 行情爆发期，放天量，大幅上涨，回调小

事有两面，盈亏同源

  • 卖飞与过山车
  • 山顶与突破
  • 风险与收益
  • 看涨不涨，看跌不跌
  • 看跌思涨，看涨思跌
  • 跟随众人，但要先于众人。

模式是基于市场认识建立的一套可重复的，能够盈利的一系列操作条件和方法。

具体包括选股条件，仓位控制，建仓周期，建仓节点，止盈条件，止损条件，加仓条件，减仓条件，平仓条件，风险防范。

契合市场状态，跟随市场资金

确定市场状态，阶段，节点，情绪，方向。市场好的时候，大干快上。市场不好，宁可不做。

寻找资金方向，只做龙头主流，不做非主流。

  • 回调、下探、观望

潜力
是不是行情初期？

大盘是否良好 -> 是否契合市场主流 -> 建仓节点是否正确 -> 尾盘建仓

跟踪主线、专注耐心、计划预案、遵循趋势，循序建仓
象杀手一样，专注耐心，趋势是会持续一段时间的，不要丢了西瓜，捡了芝麻，两头不到岸。弱水三千，只取一瓢。
建仓 （不要在第一次巨量附近建仓，对应的通常是顶。观察后续量能。）

  • 建仓节点：(预期次日买盘涌出，大趋势，20%收益预期）

    • 调整转折
    • 调整突破
    • 震荡突破
    • 上行爆发
    • 上冲下探
    • 底部启动
    • 回调支撑

  • 建仓仓位：

    一次最多不超过20%，个股仓位不超过40%，全部仓位不超过70%。

  • 建仓周期：

    尾盘一次性。

不要重复错误

  • 情绪失控，随意操作
  • 如果确定头寸错误，尽快止损。要果断，不要臆想！
  • 囿于小利，优柔寡断，成本是必须的
  • 止损位置靠近最低位置
  • 调低止损，正确的头寸通常不需要大的止损。
  • 预测顶部，不敢建仓
  • 不要在第一次巨量附近建仓，对应的通常是顶。观察后续量能。

死也要遵守纪律

  • 控制本能，摈弃没有意义的东西
  • 把事情做对，优于利益，不要囿于小利
  • 做预期内的交易
  • 非预期头寸，尽快平仓，宁可丧失头寸。
